# Compilació en C++

```bash
clang++ -std=c++17 -O2 -Wall programa.cc -o programa.exe
```

Si no es posa el `-o programa.exe` l'executable es diu `a.out`.

# Operadors aritmètics

|  |   |
| ----- | -----|
| `+` |  suma |
| `-` |  resta |
| `*` |  producte |
| `/` |  divisió entera |
| `%` |  residu divisió entera |

# Operadors relacionals

| `<=` |  menor o igual |

| `==` |  igual |
